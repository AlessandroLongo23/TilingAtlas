# T2 attack (CC-as-TA, 2026-07-06): on-grid closed, off-grid dented

**Claim (T2).** No edge-to-edge tiling by unit triangles and squares has shortest period
λ₁ ∈ (1, √3).

## On-grid case — PROVED (see th10-residual-verdict / this session)

λ₁ < 2 ⟹ one vertex per level. For λ₁ ∈ (1,√3), grid-aligned: no horizontal edge (needs λ₁|1),
no "doubling" — two edges to one level force `2·(edge horizontal) = λ₁`, i.e. λ₁ ∈ {1,√3,2}.
Enumeration: the only {3,4} vertex with distinct verticals, no horizontal, gaps ∈ {60,90} is the
tilted 4⁴. All-4⁴ ⟹ square tiling ⟹ λ₁ = 1. Contradiction. ∎

## Off-grid case — two new reductions this session

### 1. The Area Lemma (clean, general — this is the quotable one)

A fundamental domain contains `s` squares and `t` triangles, so
**det Λ = s + t·(√3/4).** Independently, det Λ = |Im(v̄₁v₂)| ∈ (1/2)ℤ[√3]. Equating the √3-parts
forces `t` even and

> **det Λ ∈ ℚ ⟺ t = 0 ⟺ the tiling is the unit square tiling ⟺ λ₁ = 1.**

Equivalently: *every triangle-bearing {3,4} tiling has irrational (√3-involving) cell area.*
(Verified: of all catalogued tilings, exactly one — the 4⁴ — has rational area.)

**What it kills.** Every off-grid period lying in ℤ[i] (the Gaussian / 45°-family) generates a
lattice of *integer* area ⟹ rational ⟹ pure square ⟹ λ₁ = 1. In particular the simplest off-grid
escape, **λ₁ = √2 via the even sublattice ⟨1+i, 1−i⟩ (area 2), is excluded.**

### 2. The 2-edge girth argument

For λ₁ ∈ (1,√3) there are no horizontal edges, so the (horizontal) period is realized by an
up/down edge path netting zero vertical. A **2-edge** realization `e₁ + e₂ = λ₁` forces
`e₁ = (h, v), e₂ = (h, −v)` with `h = λ₁/2`; both being grid edges forces them 90° apart at ±45°
to λ₁, hence **λ₁ = √2** — already killed by the Area Lemma. So any surviving off-grid
λ₁ ∈ (1,√3) needs a **≥ 3-edge girth**.

## Residual (still open)

Mixed off-grid periods — coordinates involving *both* `i` and `√3`, irrational cell area, realized
by ≥ 3-edge girths (e.g. the length-1.035 vector at 105°). The Area Lemma doesn't reach them
(irrational area), and the 2-edge argument doesn't either. Closing this needs the **thin-column
enumeration**: classify the finite ways a one-vertex-per-level column can ascend the cylinder with
grid edges and show none closes horizontally in (1,√3). That is a genuine combinatorial proof
obligation, not a one-liner.

## Honest status

- On-grid: **closed.**
- Off-grid Gaussian / `√2` / rational-area: **closed** (Area Lemma).
- Off-grid 2-edge girths: **closed** (⟹ √2 ⟹ killed).
- Off-grid mixed, ≥3-edge girths: **open** — the true residual, now sharply posed.

This is a real dent in the TA's "proof missing" item, not a closure. The Area Lemma is a clean
standalone result worth putting in the thesis regardless (it also re-proves that the only
rational-area {3,4} tiling is 4⁴). The remaining obligation is finite-combinatorial and well-defined.

## Enumeration result (th10-T2-enumerator.py, 2026-07-06)

Full finite candidate enumeration of shortest-period norms `|v|² = A+B√3 ∈ (1,3)` (coeffs ≤5):
**43 candidate norms → 3 grid-directional (killed by Part A/on-grid), 1 Gaussian √2 (killed by
Area Lemma), and 39 RESIDUAL off-grid norms.** Hermite + tile-count (`det ≥ (√3/2)|v|²`,
`det = s+t√3/4`) excludes **none** of the 39. None of the 39 occurs in the catalogue (λ₁ spectrum
jumps 1 → √3 → 1.932 → 2 …), so T2 is almost certainly true — but **each of the 39 requires a
tiling-existence exclusion; the cheap arithmetic tools leave all of them standing.**

Verdict: T2 does **not** fall to computation. The residual is substantial (≥39 concrete off-grid
directions), not a formality. Closing it needs a genuine thin-tiling existence argument (a
specialized enumeration engine) or a global structural insight — this is a real research obligation,
not a chat step.
